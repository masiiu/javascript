a) Uwagi na Temat JS:
-Kiedy powstał --> Został opracowany w 1995 roku.
-Dlaczego jest to język skryptowy --> Java Script jest jezykiem skryptowym ponieważ kod zostaje w formie tekstu.
-Gdzie jest wykonywany JS (klient lub serwer) --> Java Script działa po stronie klienta.
-Dlaczego jest to bezpieczne narzędzie --> Przeglądarka izoluje JavaScript, gdy jest wykonywany w ramach samego procesu przeglądarki. Nie może zrobić niczego, co nie jest dozwolone przez interpreter JavaScript przeglądarki lub kompilator JIT.
-Czy jest to język obiektowy , jeśli tak to dlaczego --> Java Script jest zorientowany obiektowo ponieważ w ogóle nie posiada klas.

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')" ›

type="button" - okresla przycisk.
name="przycisk" - okresla nazwe (id) przycisku.
value="Nowa Strona" - okresla tekst na przycisku.
onclick - po kliknieciu przycisku wykona sie skrypt.
